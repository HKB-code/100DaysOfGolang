## Go Learning Repository

This repository is dedicated to learning and exploring the Go programming language. It includes a variety of examples, exercises, and projects to help developers understand and master Go.

## Features

1.Comprehensive Examples: A collection of code snippets demonstrating Go’s core features and syntax.
2.Exercises: Hands-on exercises to practice and reinforce your Go skills.
3.Projects: Small projects to apply your knowledge and build real-world applications.
4.Documentation: Detailed explanations and comments within the code to aid understanding.
5.Resources: Links to additional learning materials and official Go documentation.

## Getting Started

1.Clone the Repository:
git clone https://github.com/HKB-code/100DaysOfGolang.git

2.Navigate to the Directory:
cd 100DaysOfGolang/Day1

Run the Examples:
go run test.go
